# Mingjie Li Academic Homepage

This is a static academic homepage for GitHub Pages.

## Local Preview

Open `index.html` directly in a browser, or run a local server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## GitHub Pages

For a personal GitHub Pages site, create a repository named:

```text
mlii0117.github.io
```

Push these files to the repository's `main` branch. The site will be available at:

```text
https://mlii0117.github.io
```

If using a project repository instead, enable GitHub Pages from the repository settings and publish from the `main` branch root.
